# 👁️ Multimodal Vision-Language Pipeline

[![Gemini](https://img.shields.io/badge/Gemini-1.5%20Pro-blue)](.) [![PaliGemma](https://img.shields.io/badge/PaliGemma-2-green)](.) [![Tasks](https://img.shields.io/badge/VL%20Tasks-12-orange)](.)

> Production **vision-language AI** for enterprise: chart understanding, document VQA, image captioning, visual search, grounding and video Q&A. Processes **5M+ images/month** for e-commerce and media clients.

## 🏆 Capabilities & Results
| Task | Model | Accuracy |
|------|-------|----------|
| Chart-to-Insights | Gemini 1.5 Pro | 96.2% |
| Document VQA | PaliGemma-2 fine-tuned | 91.8% |
| Product Visual Search | CLIP + custom index | 94.5% mAP |
| Video Scene QA | Gemini 1.5 Pro (1M ctx) | 88.3% |
| Image Captioning | PaliGemma-2 | BLEU-4: 0.42 |

## ✨ Enterprise Features
- **Chart understanding**: Extract data, trends and insights from any chart/graph
- **Document VQA**: Answer questions about invoices, contracts, forms
- **Visual search**: Find similar products across 50M image catalog
- **Video Q&A**: Ask questions about any point in a video (up to 2h)
- **Grounding**: Bounding box + segmentation for detected objects
